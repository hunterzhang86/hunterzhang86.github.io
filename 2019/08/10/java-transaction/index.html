本文主要介绍了几种常见的微服务分布式事务。

1. TCC

柔性事务
最终一致性
作用在服务层

2. TXC

2.1. 分布式应用里面，多条数据库记录被同时修改带来的一致性问题
fascar

3. XA

作用在资源层，依赖于数据库对XA协议的支持

4. 区别

4.1. 概要

4.2. XA是资源层面的分布式事务，强一致性，在两阶段提交的整个过程中，一直会持有资源的锁
4.3. TCC是业务层面的分布式事务，最终一致性，不会一直持有资源的锁
4.4. XA一般对应用开发者屏蔽
4.5. TCC没有对开发者屏蔽，开发者还需要自己实现接口
