Java 基元简介

 2023.1.20  2023.5.12  手册  2323  5 分钟  

一、概述

Java 编程语言具有八种原始数据类型。

在本教程中，我们将了解这些原语是什么，并介绍每种类型。

2.原始数据类型

Java 中定义的八个原语是 int 、 byte 、 short 、 long 、 float 、 double 、 boolean 和 char。这些不被视为对象并代表原始值。

它们直接存储在堆栈中（查看 这篇文章 了解有关 Java 内存管理的更多信息）。

我们将看一下存储大小、默认值以及如何使用每种类型的示例。

让我们从一个快速参考开始：

|类型 |大小（位）|最低 |最大 |示例 | |-----------|-------------|------------|----------- ---------|---------------------------------| | 字节 | 8 | -2^7^ | 2^7^-- 1 | 字节 b = 100; | | 短 | 16 | -2^15^ | 2^15^-- 1 | 短 s = 30_000; | | 整数 | 32 | -2^31^ | 2^31^-- 1 | int i = 100_000_000; | | 长 | 64 | -2^63^ | 2^63^-- 1 | 长 l = 100_000_000_000_000; | | 浮动 | 32 | -2^-149^ | (2-2^-23^)·2^127^ | float f = 1.456f; | | 双 | 64 | -2^-1074^ | (2-2^-52^)·2^1023^ | 双f = 1.456789012345678; | | 字符 | 16 | 0 | 2^16^-- 1 | char c = 'c'; | | 布尔 | 1 | -- | -- | 布尔b =真； |

2.1。 整数

我们要介绍的第一个原始数据类型是 int 。也称为整数，int 类型包含范围广泛的非小数数值。

具体来说，**Java 使用 32 位内存存储它。**换句话说，它可以表示从 -2,147,483,648 (-2^31^) 到 2,147,483,647 (2^31^-1) 的值。

在 Java 8 中，可以通过使用新的特殊辅助函数来存储最大为 4,294,967,295 (2^32^-1) 的无符号整数值。

我们可以简单地声明一个 int：

1
2
3

诠释 x = 424_242;

整数y;

在没有赋值的情况下声明的 int 的默认值为 0。

如果变量是在方法中定义的，我们必须先赋值才能使用它。

我们可以对 int s 执行所有标准算术运算。请注意，在对整数执行这些操作时，十进制值将被截断。

2.2。 字节

byte 是一种类似于 int 的原始数据类型，除了它只占用 8 位内存。 这就是我们称它为字节的原因。因为内存太小，byte 只能保存-128 (-2^7^) 到127 (2^7^-- 1) 的值。

以下是我们创建 byte 的方法：

2
3

字节 b = 100;

字节空；

** byte 的默认值也是 0。**

2.3。 短的

我们的 Java 原始数据类型列表的下一站是 short。

如果我们想节省内存并且 byte 太小，我们可以使用 byte 和 int 之间的类型：short。

在 16 位内存中，它是 int 大小的一半，是 byte 大小的两倍。它的可能值范围是 -32,768(-2^15^) 到 32,767(2^15^-- 1)。

short 声明如下：

1
2
3

短 s = 20_020;

短裤;

也与其他类型类似，默认值为 0。我们也可以在其上使用所有标准算术。

2.4。 长

我们最后一个与整数相关的原始数据类型是 long。

long 是 int 的大哥。 它存储在 64 位内存中，因此它可以保存更大的可能值集。

long 的可能值介于 -9,223,372,036,854,775,808 (-2^63^) 到 9,223,372,036,854,775,807 (2^63^-- 1) 之间。

我们可以简单地声明一个：

1
2
3

长 l = 1_234_567_890;

长 l;

与其他整数类型一样，默认值也是 0。我们可以在 long 上使用所有适用于 int 的算术。

2.5。 漂浮

我们使用 float 类型表示 Java 中的基本小数。这是一个单精度十进制数。这意味着，如果我们超过了小数点后六位，这个数字就会变得不那么精确，而更像是一个估计值。

在大多数情况下，我们并不关心精度损失。但是如果我们的计算需要绝对的精度（例如，金融操作、登月等），我们需要使用为这项工作设计的特定类型。有关更多信息，请查看 Java 类 Big Decimal。

此类型与int一样存储在32位内存中。 但是，由于是浮点数，其范围有很大不同。它可以表示正数和负数。最小的小数是 1.40239846 x 10^-45^，最大值是 3.40282347 x 10^38^。

我们声明 floats 与任何其他类型相同：

1
2
3

浮动 f = 3.145f；

浮动 f;

默认值是 0.0 而不是 0。 另外，请注意我们将 f 指定添加到文字数字的末尾以定义浮点数。否则，Java 将抛出错误，因为十进制值的默认类型是 double。

我们还可以对 floats 执行所有标准算术运算。然而，重要的是要注意我们执行浮点运算与整数运算非常不同。

2.6。 双倍的

接下来，我们看看 double。它的名字来源于它是一个双精度十进制数。

它存储在 64 位内存中。 这意味着它代表的可能数字范围比 float 大得多。

虽然，它确实受到与 float 相同的精度限制。范围是 4.9406564584124654 x 10^-324^ 到 1.7976931348623157 x 10^308^。该范围也可以是正数或负数。

声明 double 与其他数字类型相同：

1
2
3

双 d = 3.13457599923384753929348D；

双 d;

默认值也是 0.0，因为它与 float . 类似 float， 我们附加字母 D 以将文字指定为双精度值。

2.7。 布尔

最简单的原始数据类型是 boolean 。它只能包含两个值： true 或 false 。 它将其值存储在一个位中。

然而，为方便起见，Java 填充该值并将其存储在单个字节中。

以下是我们声明 boolean 的方式：

1
2
3

布尔 b = 真；

布尔值 b;

在没有值的情况下声明它默认为 false 。 boolean 是控制我们的程序流的基石。我们可以对它们使用布尔运算符（例如，and、or 等）。

2.8。 字符

最后要查看的原始数据类型是 char。

char 也称为字符，是一个 16 位整数，表示 Unicode 编码的字符。它的范围是从 0 到 65,535。在 Unicode 中，这表示 '\u0000' 到 '\uffff'。

有关所有可能的 Unicode 值的列表，请查看 Unicode 表 等网站。

现在让我们声明一个 char：

在定义变量时，我们可以使用任何字符文字，它们会自动转换为我们的 Unicode 编码。字符的默认值为 '/u0000'。

2.9。溢出

原始数据类型有大小限制。但是如果我们尝试存储一个大于最大值的值会发生什么？

我们遇到了一种称为 overflow 的情况。

当整数溢出时，它会翻转到最小值并从那里开始向上计数。

浮点数通过返回 Infinity 溢出：

1
2
3
4
5
6
7

int i = Integer.MAX_VALUE;
整数 j = i + 1;
// j 将翻转到 -2_147_483_648

双 d = 双.MAX_VALUE；
双 o = d + 1;
// o 将是无穷大

下溢是相同的问题，只是它涉及存储一个小于最小值的值。当数字下溢时，它们返回 0.0。

在本文中，我们介绍了 Java 支持的八种原始数据类型。

这些是大多数（如果不是全部）Java 程序使用的构建块，因此非常值得了解它们是如何工作的。
