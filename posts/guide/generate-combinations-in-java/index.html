在 Java 中生成组合

 2023.3.21  2023.4.3  手册  3531  8 分钟  

一、简介

在本教程中，我们将讨论 Java 中 k-combinations 问题的解决方案。

首先，我们将讨论并实现递归和迭代算法，以生成给定大小的所有组合。然后我们将回顾使用通用 Java 库的解决方案。

2.组合概述

简而言之，组合是给定集合中元素的子集。

与排列不同，我们选择单个元素的顺序无关紧要。相反，我们只关心特定元素是否在选择中。

例如，在纸牌游戏中，我们必须从由 52 张牌组成的牌组中发 5 张牌。我们对选择 5 张卡片的顺序不感兴趣。相反，我们只关心手中有哪些牌。

有些问题需要我们评估所有可能的组合。为了做到这一点，我们列举了各种组合。

因此，在最坏的情况下，选择元素的方法的数量会呈指数增长。因此，对于大量人口，可能无法列举不同的选择。

在这种情况下，我们可能会随机选择一些有代表性的选择。该过程称为采样。

接下来，我们将回顾各种算法来列出组合。

  1. 生成组合的递归算法

[递归算法]（/java-recursion）通常通过将问题划分为类似的较小问题来工作。这个过程一直持续到我们达到终止条件，这也是基本情况。然后我们直接解决基本情况。

我们将讨论两种方法来细分从集合中选择元素的任务。 第一种方法根据集合中的元素来划分问题。第二种方法通过仅跟踪所选元素来划分问题。

3.2。按组合中的元素划分

我们将通过跟踪选择中的项目来划分任务，而不是跟踪输入集中的元素。

  1. 迭代算法

在迭代方法中，我们从初始组合开始。然后，我们继续从当前组合生成下一个组合，直到我们生成所有组合。

让我们按字典顺序生成组合。我们从最低的词典组合开始。

为了从当前组合中获得下一个组合，我们在当前组合中找到可以递增的最右边的位置。然后，我们增加位置并在该位置右侧生成可能的最低字典组合。
